# CADI SHOP

CADI SHOP is a professional e-commerce platform designed for selling beauty, grooming, and personal care products online. The website provides customers with a seamless shopping experience while giving store administrators powerful tools to manage products, categories, orders, and store content.

🌐 Live Website: https://cadishop1.netlify.app/

## Features

* Modern and responsive design
* Product catalog and category management
* Shopping cart and checkout system
* Order management dashboard
* Store customization tools
* Testimonials and promotional sections
* Cloud database powered by Supabase
* Fast and reliable hosting with Netlify
* Mobile-friendly user experience

## Technology Stack

* React
* TypeScript
* Vite
* Supabase
* Netlify

## Live Demo

https://cadishop1.netlify.app/

## Purpose

This repository contains the source code for the CADI SHOP online store, developed to provide a modern, scalable, and user-friendly e-commerce solution for real-world business operations.
